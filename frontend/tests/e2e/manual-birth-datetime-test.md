# 生年月日時表示の手動テスト手順

## 目的
入力した生年月日時が、詳細ページとリストページで正しく表示されることを確認

## テストケース

### 事前準備
1. ブラウザの開発者ツール（F12）を開く
2. Consoleタブで以下を実行してLocalStorageをクリア：
   ```javascript
   localStorage.clear()
   ```
3. ページをリロード

### テスト手順

#### Step 1: トップページで生年月日時を入力
1. `http://localhost:3248` にアクセス
2. 名前を入力：`テスト太郎`
3. 生年月日を選択：`1986年5月26日`
4. 時刻を選択：`05:00`（朝5時）
5. 性別を選択：`男性`
6. 「命式を計算」ボタンをクリック

#### Step 2: 詳細ページで表示確認
- 基本情報欄に **「1986年5月26日 5:00」** と表示されることを確認
- ❌ 「1986年5月25日 20:00」と表示される場合は、タイムゾーン変換が正しく動作していません

#### Step 3: リストページで表示確認
1. ヘッダーのリストアイコンをクリック
2. 命式カードに **「1986年5月26日 5:00」** と表示されることを確認
- ❌ 「1986年5月25日 20:00」と表示される場合は、タイムゾーン変換が正しく動作していません

### デバッグ方法

開発者ツールのConsoleで以下を実行して、保存されているデータを確認：

```javascript
// LocalStorageのデータを確認
const data = JSON.parse(localStorage.getItem('saju_data') || '[]');
console.log('LocalStorage data:', data);

// birthDatetimeを確認
if (data.length > 0) {
  console.log('birthDatetime:', data[0].birthDatetime);
}
```

### 期待される結果
- birthDatetime: `1986-05-26T05:00:00+09:00` （またはこれに類する形式）
- 詳細ページ表示: `1986年5月26日 5:00`
- リストページ表示: `1986年5月26日 5:00`

### 問題が発生した場合
1. ブラウザのConsoleでエラーを確認
2. Network タブで API レスポンスの `birthDatetime` を確認
3. `formatBirthDateTime` 関数の動作をデバッグ
