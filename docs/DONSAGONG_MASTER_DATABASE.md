## 🚨 절대 원칙

### ❌ 절대 금지사항
- **오행론**: 목화토금수, 상생상극 사용 금지
- **십신론**: 전통 명리학의 십신 개념 사용 금지
- **신강/신약론**: 일간의 강약 판단 금지
- **전통 합충**: 전통 명리학의 합충 해석 금지
- **년주 사용**: 돈사공은 3주 체계

### ✅ 돈사공 고유 원칙
- **용신 = 무기** (부족을 채우는 게 아님)
- **일지지 합은 무조건 길** 
- **조후용신 80% : 원국 20%** (대운이 4배 중요)
- **천간합(合)은 기본적으로 흉** 
- **월지지는 용신 불가**

#### STEP 1: 만세력 계산 (사주 원국 + 대운)
```python
# 반드시 먼저 해야 할 것
from datetime import datetime
from src.manseryeok.calculator import ManseryeokCalculator

# 1. 생년월일시와 성별 확인
birth_datetime = datetime(년, 월, 일, 시, 분, 초)
gender = 'male' or 'female'  # 성별 필수!

# 2. 사주 계산
calculator = ManseryeokCalculator()
saju = calculator.calculate_saju(birth_datetime, gender)

# 3. 원국 확인
print(f'년주: {saju.year_stem}{saju.year_branch}')
print(f'월주: {saju.month_stem}{saju.month_branch}')
print(f'일주: {saju.day_stem}{saju.day_branch}')
print(f'시주: {saju.hour_stem}{saju.hour_branch}')
print(f'일간: {saju.day_stem}')  # 가장 중요!

# 4. 대운 계산
daeun_info = calculator.calculate_daeun_with_lunar(saju)
```

#### STEP 2: 원국 분석
```
1. 일간 확인 → 모든 분석의 기준
2. 월지 확인 → 계절 판단 (조후용신)
3. 천간 구성 파악
4. 지지 구성 파악
```

#### STEP 3: 원국 용신 추출 (대운과 다름!)

##### 【용신이란】
- **용신**: 일간, 지지가 (몸)이라면, 내가 세상을 살아가는 "무기"가 용신
- **특징**: 용신을 안 쓰다고 모르는 것이 아니다
- **활용**: 사주에서의 최상인 천간, 지지와의 구성을 이해하여 최고의 사주 실현

##### 【힘의 순서 (찾는 순서)】
```
1. 용신 이용 = 천간(투간) - 월간 A, 일천간 B
2. 흐름 이용 = 지지 - 일지 C, 시지 D (조후 용신법)

※ 1,2번 순으로 다 있을수록 훨씬 좋다
```

##### 【용신 위치와 의미】
```
천간 용신 (남녀 공통 사용 가능):
- A: 월간 = 일의 능력, 학업 능력, 오래가기
- B: 시간 = 일의 센스, 처세술, 빠른 습득

지지 용신 (성별 차이 있음):
- C: 일지 = 가정/배우자/건강/귀인/투자
  * 남자: 모두 활용 가능
  * 여자: 일 또는 가정 중 선택 (아이가 다 컸거나 없으면 둘 다 가능)
- D: 시지 = 일의 결과/성과
  * 남자: 모두 활용 가능
  * 여자: 일 또는 가정 중 선택 (아이가 다 컸거나 없으면 둘 다 가능)

❌ 불가능한 위치:
- 년주 전체
- 월지

⚠️ 주의: 원국 용신과 대운 해석은 완전히 다름!

🔥 중요 인사이트:
- 일지지 = 개인적 영역 (가정, 배우자, 귀인, 건강, 투자 등)
- 시지지 = 일의 결과/성과 영역
- 정화(丁)/신금(辛) 일간만 일과 건강이 세트로 연결됨
```

#### STEP 4: 조후 분석
```
1. 원국 월지 확인 (子=12월, 丑=1월...)
2. 해당 월의 조후길흉표 확인
3. 특수 일간 체크:
   - 정화(丁), 신금(辛): 일과 건강이 세트로 연결됨
   - 다른 일간: 일은 시지, 건강은 일지로 구분

⚠️ 조후 사용처:
- 원국 월지 조후: 원국 용신 찾을 때만 사용
- 대운 지지 조후: 대운 길흉 판단시 사용 (나한테 유리한 조후가 들어오는지)

🔄 원국 용신 vs 대운 용신:
- 원국 용신: 원국 월지 계절에 맞는 용신
- 대운 용신: 대운 지지 계절에 맞는 용신 (대운마다 바뀔)
- 둘 다 사용 가능, 여자는 지지 용신을 일/가정 중 선택
```

#### STEP 5: 대운/세운 정확한 해석법 ⭐⭐⭐

##### 【핵심 원칙】
**일주와 시주에만 대운/세운을 대입한다! (년주/월주는 보지 않음)**

⭐ **일주의 중요성**: 
- 일주의 운이 가장 중요함
- 시주 운이 떨어져도 일주 운이 강하고 좋으면 일이 잘 풀림
- 일주가 개인의 핵심 운세를 결정하는 반증

##### 【4포인트 체크 시스템】
```
대운/세운이 들어오면 4개 관계를 체크:
1. 일간 vs 운의 천간
2. 일지 vs 운의 지지  
3. 시간 vs 운의 천간
4. 시지 vs 운의 지지

목표: 4개 중 길이 3개 이상 = 대박!
```

##### 【길흉 판단 기준】
```
천간 관계:
- 합(合): 흉
- 충(沖): 흉 또는 길
- 비견(같은 천간): 흉
- 천간 매트릭스 참조

지지 관계:
- 합(合): 길 ✅
- 충(沖): 흉 ❌
- 지지 매트릭스 참조
```

##### 【조후 용신법 - 3단계 판단】
```
✅ 체크 1: 내 지지가 운의 지지와 길?
   → 지지끼리 삼합 관계인가
   → 일지: 개인생활/건강 좋다
   → 시지: 일의 결과 좋다

✅ 체크 2: 내 천간이 운의 천간과 길?  
   → 천간끼리 길인가
   → 판단/계획이 좋다

✅ 체크 3: 조후 판단
   → 원국 월지의 반대 계절이 대운/세운으로 들어오는가?
   → 일지 조후: 개인생활/건강에 영향
   → 시지 조후: 일의 결과에 영향
   
※ 특수규칙:
- 보통: 월지가 여름이면 겨울대운이 좋음
- 정화(丁)/신금(辛) 일간: 월지가 겨울이어도 겨울 조후가 일/건강에 좋음
```

#### STEP 6: 종합 판단
```
최종 평가:
- 4포인트 중 길 3-4개: 대길 (상)
- 4포인트 중 길 2개: 보통 (중)
- 4포인트 중 길 0-1개: 주의 (하)

+ 조후 추가 고려:
  - 일지 조후 길/흉 → 개인생활/건강 영향
  - 시지 조후 길/흉 → 일의 결과 영향
  
⚠️ 중요: 일주 운이 가장 중요함
  - 시주 운이 나빠도 일주 운이 좋으면 일이 잘 풀림
```

## 3️⃣ 조후용신 시스템

> 계절의 온도와 습도를 조절하는 공통원국지지용신
> 
> 출처: 학습자료 대운.067-070.png, 용신.005-006.png

### ⚠️ 중대 구분: 원국 조후용신 vs 대운 조후용신
**원국의 조후용신과 대운의 조후용신은 완전히 다른 시스템입니다!**

### 🔥 특수 일간: 정화(丁火)와 신금(辛金)
**정화(丁火)와 신금(辛金)은 월지지가 아닌 일간으로 조후를 봐야 하는 특수한 경우입니다!**

- **정화(丁火) 일간**: 가을, 겨울 조후가 좋음 (월지지 무관)
- **신금(辛金) 일간**: 가을, 겨울 조후가 좋음 (월지지 무관)
- **나머지 8개 일간**: 월지지로 조후 판단

#### 1. 원국의 일시지지 조후 맞추기 (용신.006.png)
- **내가 노력으로 환경을 맞춰야 하는 것** (결과)
- **원국의 일시지지에 무조건 따뜻한 글자가 있어야 함**
- **최고의 따뜻함 글자로 지지않을 수 있음: 丁, 戊, 午 필수**
- **가을, 겨울생일 때는 丙, 丁를 피하려해야 지나 곰이 됨: 냉하게 한다. 寅午戌 필수**

#### 2. 계절별 조후 현실화 가이드 (용신.005.png)

**계절별 조후 원칙:**
- **봄, 2,3,4월 생**: 봄, 여름이 좋음. 가을,겨울까지
- **여름, 5,8,9,10월 생**: 봄, 여름이 좋음. 가을 상냄 잘 전까지
- **겨울, 11,12,1월 생**: 봄, 여름이 좋음. 가을,겨울까지

**지지에 봄여름 글자가 있으면 봄, 여름에 좋음.**

**그래서 甲, 乙 당이 시작하던 봄지는 다 깨지기 시작해서 조후이 없음.**

**가을 겨울이 되면 甲, 戊 당이 시작하면 봄지는 다 깨지기 시작해서 조후이 없음.**

**가을 겨울은 지지 용신이 아해지거나 몇 없음.**

### 조후용신의 기본 원리
- **계절의 온도와 습도 조절**이 핵심
- **너무 뜨거우면 차갑게, 너무 차가우면 따뜻하게**
- **너무 건조하면 축축하게, 너무 습하면 말려야 함**

>

### 용신의 정의
- **용신 = 무기** (부족을 채우는 것이 아님)
- **천간 용신**: 천간에서 찾는 용신
- **지지 용신**: 지지에서 찾는 용신

### 🚨 용신 추출 원칙
- ❌ **년주에서 용신 추출 금지**
- ❌ **월지지는 용신 불가**
- ✅ **월간, 일간, 일지, 시간, 시지에서만 용신 추출 가능**

### 📋 출생시간 미상시 대응
- **시간을 아는 경우**: 월간, 일간, 일지, 시간, 시지 (5개 위치)
- **시간을 모르는 경우**: 월간, 일간, 일지 (3개 위치만 사용)
- **시주는 제외하고 월주와 일주로만 용신 분석**

### 🚫 해석 제외 원칙
- **시간 미상시**: 시주에서 알 수 있는 모든 정보는 해석에서 제외

### 🎯 DONSAGONG 분석 체크리스트

#### 📝 분석 순서
1. **만세력 계산** → 사주 + 대운 확정
2. **용신 추출** → 월간, 일지, (시간, 시지) 확인
3. **조후 분석** → 월지별 대운 길흉 판단
4. **천간 100매트릭스** → 일간 vs 대운천간
5. **지지 144매트릭스** → 월지/일지 vs 대운지지
6. **종합 판단** → 용신+조후+매트릭스 통합

#### 🔍 용신 체크포인트
- ✅ 월간에서 찾기 (월지 제외)
- ✅ 일지에서 찾기
- ✅ 시간/시지에서 찾기 (시간 아는 경우)
- ❌ 년주 절대 금지
- ❌ 월지지 금지

#### 🌡️ 조후 빠른 확인
**여름생(巳午未)**: 겨울 대운 최고
**봄생(寅卯辰)**: 여름 대운 좋음
**가을생(申酉戌)**: 여름 대운 좋음  
**겨울생(亥子丑)**: 여름 대운 최고

#### ⚠️ 특수 일간 주의
- **정화(丁)·신금(辛)**: 일운/건강운만 특수 조후

### 핵심 암기 사항
1. **용신 = 무기** (부족을 채우는 게 아님)
2. **일지지 합은 무조건 길** 
3. **조후용신 80% : 원국 20%** (대운이 4배 중요)
4. **천간합은 기본적으로 흉** 
5. **월지지는 용신 불가**
6. **년주에서 용신 추출 금지**

### 🔥 대운 판단 우선순위
1. **조후 확인** (가장 중요)

3. **매트릭스 길흉**
4. **합충 관계**

### 💡 실전 팁
- 시간 모를 때: 월간, 일지만 사용
- 조후 나쁘면: 다른 요소 좋아도 주의
- 용신 있으면: 기본적으로 유리
- 대흉 매트릭스: 피하는 것이 상책
- **일주 운이 가장 중요**: 시주보다 일주의 영향력이 크다
